JS中遍历

发表于 2018-09-06 |

JS中数组遍历除了常用的for语句，还有map, some, every, forEach，都可以用作循环，但是他们又有各自的用法。

forEach没有返回值，只针对每个元素调用func。

map有返回值，返回一个新的数组，每个元素为调用func的结果。

forEach和map都会遍历所有选项，中间不能跳出循环。当想跳出循环可以使用 every 和 some 方法，下面是简单的总结.

every 当内部 return false时跳出整个循环（return true;也是需要写明，不写的话默认为retrun undefined ）

some 当内部return true时跳出整个循环
